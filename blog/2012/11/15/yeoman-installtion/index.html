    YEOMAN をインストールしてみる

    巷で噂のYEOMANをインストールしてみる。

    YEOMANとは?

    公式サイトによると、

    • Package Management : パッケージ管理
    • Build System : ビルドタスク
    • LiveReload Watch Process : LiveReload
    • Generators : scaffolding
    • Local Server

    と、最近のWebアプリでよく用いられるライブラリ・ツールをまとめて管理してくれるもの。 対話式コンソールでBackbone.js やら Require.js やらのライブラリを選択していくと、 基本的な骨組みが簡単に生成されるCUIツール(のはず)。
